Link nodes

Well done, your device has been created!

If you already have installed the fabric on your nodes you can link them to {{ device.name }} by running the following command on the commandline:

your-device # bb link {{ device.security_code }}

Once your node is linked to {{ device.name }} it can be used to provision tints to it.

You can also choose to skip this step and add nodes to {{ device.name }} at some later point. You can find the security_code to use for adding devices to your hex on in your device list. ( It is {{ device.security_code }}, in case you want to write it down. )
